# What Do You Want For Dinner?

This will eventually be a collection of scripts that can be run when my partner asks me what I want
for dinner. Currently the only script is `pick-dinner.py` which randomly selects meals that we
usually make at home for me to suggest.

## Planned additional scripts:
- eat-out
  - A tool that grabs the nearest restaurants and picks one based on ratings and other metrics like
  "my taste" or cost.
- This will be an expanding list...
